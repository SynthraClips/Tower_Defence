# /goal 6 — Projectiles and Combat Feedback

```text
/goal Improve projectile behaviour and combat feedback for the four tower archetypes.

Focus only on combat/projectiles and hit feedback.

Tower archetypes:
- Light attack
- Heavy attack
- Magic-type attack
- Air attack

Tasks:
- Match projectile style to each tower archetype.
- Add light attack projectile behaviour.
- Add heavy attack projectile behaviour.
- Add magic projectile behaviour.
- Add air-targeting projectile behaviour.
- Ensure air attack towers use hooks that can later support flying/air enemies.
- Add clearer hit feedback on enemies.
- Add splash/water impact feedback where appropriate.
- Make damage feedback readable without needing final art.
- Keep combat code modular and easy to expand.

Do not rebalance all waves or build the full upgrade system in this goal.

Acceptance criteria:
- Each tower archetype fires a distinct projectile.
- Hits are visually readable.
- Water/splash impact feedback exists where appropriate.
- Combat remains stable across multiple enemies and towers.
```
