# /goal 9 — Art and Presentation Pass

```text
/goal Create a coherent placeholder art and presentation pass for the water-based tower defence game.

The aim is not final production art yet, but the game should feel visually consistent and readable.

Focus only on art, visual identity, and presentation consistency.

Tasks:
- Create or improve enemy boat sprites/silhouettes.
- Create or improve tower sprites for all four tower types:
  - Light
  - Heavy
  - Magic
  - Air
- Improve water route visuals.
- Improve land / non-water buildable tile visuals.
- Improve harbour / base / core visual.
- Improve route marker visuals.
- Improve UI graphics consistency.
- Make the visual style feel like one coherent project.
- Keep assets organised under the existing project-owned asset folders:
  - Assets/Art
  - Assets/Prefabs
  - Assets/Audio where relevant

Do not rewrite gameplay systems in this goal unless required to hook up assets.

Acceptance criteria:
- Boats, towers, water, land, route, and base are visually distinct.
- The project has a more coherent visual identity.
- Placeholder art is good enough for testing and iteration.
- Asset organisation remains clean.
```
