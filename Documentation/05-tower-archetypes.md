# /goal 5 — Tower Archetypes

```text
/goal Replace placeholder tower identity with the four intended tower archetypes.

The current tower classes should be themed and structured around:
- Light attack
- Heavy attack
- Magic-type attack
- Air attack

Focus only on tower data, identity, placement integration, and basic behaviour hooks.

Tasks:
- Create or update tower definitions for:
  - Light attack tower
  - Heavy attack tower
  - Magic-type tower
  - Air attack tower
- Give each tower:
  - Range
  - Rate of fire
  - Cost
  - Sell ratio
  - Damage type
  - Cooldown
  - Upgrade path hook
  - Target preference hooks
  - Attack preference modes: close / weak / far
- Make tower visuals clearly different using placeholder art if needed.
- Ensure all towers can be placed on valid non-water tiles.
- Ensure towers use the existing gold/cost flow.
- Keep the tower system expandable for future upgrades and targeting improvements.

Do not fully build upgrade UI, final art, or full combat VFX in this goal.

Acceptance criteria:
- Four tower archetypes exist and can be placed.
- Each tower has different gameplay stats.
- Each tower has clear placeholder visual identity.
- Tower data is structured for future upgrades and targeting modes.
```
