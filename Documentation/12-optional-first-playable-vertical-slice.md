# Optional — First Playable Vertical Slice

```text
/goal Build the first playable vertical slice of the water-based tower defence game.

Use the completed systems from the previous goals and bring them together into one polished first-playable version.

The vertical slice should include:
- One readable river/harbour map.
- Water route based on waypoints.
- Open land / non-water tower placement.
- Clear spawn point.
- Clear base/end point.
- Four tower archetypes:
  - Light
  - Heavy
  - Magic
  - Air
- First enemy boat ladder:
  - Weak
  - Fast weak/raft if useful
  - Medium
  - Hard
  - Boss
- First five balanced waves.
- Gold, lives, wave, state, victory, and game over HUD.
- Money spent feedback.
- Lives lost feedback.
- Basic combat feedback.
- Placeholder but coherent art.
- Basic audio pass.
- Stable restart/game-over/victory flow.

Acceptance criteria:
- The game is playable from start to finish.
- The player understands where enemies come from and where they are going.
- The player can place towers on valid land.
- Waves escalate in difficulty.
- The first boss wave feels like a milestone.
- The game has enough visual and audio feedback to judge whether the core loop is fun.
- The codebase remains clean enough to keep expanding.
```
