# Release Readiness Checklist

Use this checklist before cutting a build for the water tower defence ship window.

- [ ] Build Settings scenes checked: `MainMenu`, `LevelSelect`, `Settings`, `Easy Level`, `MediumLevel`, `HardLevel`
- [ ] `MainMenu -> LevelSelect -> Easy Level` tested
- [ ] `MainMenu -> LevelSelect -> MediumLevel` tested
- [ ] `MainMenu -> LevelSelect -> HardLevel` tested
- [ ] Direct play from `Easy Level` tested in the Unity Editor
- [ ] Direct play from `MediumLevel` tested in the Unity Editor
- [ ] Direct play from `HardLevel` tested in the Unity Editor
- [ ] Audio settings sliders tested with mixer assigned
- [ ] Pause and resume tested with keyboard and UI buttons
- [ ] Victory flow tested with UI buttons
- [ ] Defeat flow tested with UI buttons
- [ ] Restart Level button tested from pause, victory, and defeat states
- [ ] Level Select button tested from pause, victory, and defeat states
- [ ] Main Menu button tested from pause, victory, and defeat states
- [ ] Tower placement tested on valid land
- [ ] Invalid tower placement tested on water/path and blocked areas
- [ ] Every tower type tested at least once in a live run
- [ ] Every enemy type tested at least once in a live run
- [ ] Easy level path visual checked for width/alignment
- [ ] Medium level path visual checked for width/alignment
- [ ] Hard level path visual checked for width/alignment
- [ ] Boats visually face their direction of travel
- [ ] Projectiles spawn from the intended fire points
- [ ] No console errors during one full run of each gameplay scene
- [ ] `Tools/Tower Defence/Validate Game Setup` run and reviewed
- [ ] `Tools/Tower Defence/Wave Balance Report` run and reviewed
- [ ] Windows build created successfully
- [ ] Built executable launched outside the Unity Editor
- [ ] Known issues reviewed and logged below
