# /goal 10 — Audio Pass

```text
/goal Add the first complete audio pass for the tower defence game.

Focus only on audio assets, playback hooks, and basic mixing.

Tasks:
- Add placement sound.
- Add light tower fire sound.
- Add heavy tower fire sound.
- Add magic tower fire sound.
- Add air tower fire sound.
- Add boat hit sound.
- Add boat death sound.
- Add water/splash impact sound.
- Add base hit sound.
- Add round start sound.
- Add round complete sound.
- Add a background loop.
- Make sure sounds are not painfully loud or overlapping badly.
- Keep audio assets organised under Assets/Audio.
- Add simple volume/mixer hooks if appropriate.

Do not rebalance gameplay or rework visuals in this goal.

Acceptance criteria:
- Core gameplay actions have sound feedback.
- The background loop plays correctly.
- Important sounds are audible but not overpowering.
- Audio files are cleanly organised.
```
